üöÄ Quick Deploy

Method 1: Vercel CLI

  1. Install Vercel CLI bash npm i -g vercel

  2. Login to Vercel bash vercel login

  3. Deploy bash vercel

  4. Deploy to Production bash vercel --prod

Method 2: Git Integration (Recommended)

  1. Push to GitHub bash git add . git commit -m "Deploy FastAPI to Vercel" git push origin master

  2. Import in Vercel

  3. Go to vercel.com
  4. Click "Add New" ‚Üí "Project"
  5. Import your GitHub repository

  6. Vercel will auto-detect the configuration

  7. Deploy

  8. Click "Deploy"
  9. Vercel will build and deploy automatically

src/api/index.py

This file is the entry point for Vercel. It imports the FastAPI app from main.py.

üêõ Troubleshooting

Build Fails

Problem: Build fails with module import errors

Solution: Make sure all dependencies are in requirements.txt

pip freeze > requirements.txt

404 Errors

Problem: All routes return 404

Solution: Check that src/api/index.py properly imports the app:

from main import app

Function Timeout

Problem: Requests timeout after 10 seconds

Solution: Vercel free tier has 10s timeout. Upgrade or optimize code.

Static Files Not Loading

Problem: HTML template not found or "Template file not found" error

Solution:

  1. Ensure public/ directory is included in the deployment:
  2. The vercel.json should include static build configuration

  3. Check that public/home.html exists in the repository

  4. The application now supports multiple path resolution strategies:

  5. /var/task/public/home.html (Vercel serverless)
  6. Relative paths for local development

  7. Fallback to embedded HTML if template not found

  8. Verify environment variables are set: bash ENVIRONMENT=production API_VERSION=2.0.0

  9. If the issue persists, check Vercel build logs for file inclusion.

üîÑ CI/CD

Vercel automatically deploys:

  • Production: Pushes to main or master branch
  • Preview: Pushes to any other branch or pull requests

Disable Auto-Deploy

In Vercel Dashboard: 1. Project Settings ‚Üí Git 2. Disable "Production Branch Auto Deploy" 3. Manually trigger deployments

üîê Security

Environment Variables

    CORSMiddleware,

    allow_credentials=True,
    allow_methods=["GET", "POST"],
    allow_headers=["*"],
